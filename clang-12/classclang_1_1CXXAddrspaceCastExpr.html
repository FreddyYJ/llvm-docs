Detailed Description

A C++ addrspace_cast expression (currently only enabled for OpenCL).

This expression node represents a cast between pointers to objects in different address spaces e.g., addrspace_cast<global int*>(PtrToGenericInt).

A addrspace_cast can cast address space type qualifiers but does not change the underlying value.
