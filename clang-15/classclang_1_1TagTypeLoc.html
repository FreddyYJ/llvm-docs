Detailed Description

Wrapper for source info for tag types.

Note that this only records source info for the name itself; a type written 'struct foo' should be represented as an ElaboratedTypeLoc. We currently only do that when C++ is enabled because of the expense of creating an ElaboratedType node for so many type references in C.
