◆ internalizeFunction()

static Function * internalizeFunction ( Function & F)
static

Make another copy of the function F such that the copied version has internal linkage afterwards and can be analysed.

Then we replace all uses of the original function to the copied one

Only non-exactly defined functions that have linkonce_odr or weak_odr linkage can be internalized because these linkages guarantee that other definitions with the same name have the same semantics as this one
