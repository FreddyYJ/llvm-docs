Detailed Description

The BTF common type definition.

Different kinds may have additional information after this structure data.

◆ [union]

union { ... } llvm::BTF::CommonType

"Size" is used by INT, ENUM, STRUCT and UNION.

"Size" tells the size of the type it is describing.

"Type" is used by PTR, TYPEDEF, VOLATILE, CONST, RESTRICT, FUNC, FUNC_PROTO and VAR. "Type" is a type_id referring to another type.

uint32_t llvm::BTF::CommonType::Info

"Info" bits arrangement: Bits 0-15: vlen (e.g.

# of struct's members) Bits 16-23: unused Bits 24-27: kind (e.g. int, ptr, array...etc) Bits 28-30: unused Bit 31: kind_flag, currently used by struct, union and fwd
