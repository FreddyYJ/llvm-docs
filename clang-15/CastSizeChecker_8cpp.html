Check if we are casting to a struct with a flexible array at the end.

struct foo {
size_t len;
struct bar data[];
};

or

struct foo {
size_t len;
struct bar data[0];
}

In these cases it is also valid to allocate size of struct foo + a multiple of struct bar.
