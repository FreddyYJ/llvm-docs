Detailed Description

Models a class member access.

C++ [expr.ref] member-expression: expression -> template_opt id-expression expression . template_opt id-expression e.g. x.a, xp->a

Note: An implicit member access inside a class, i.e. a instead of this->a, is an id-expression.
