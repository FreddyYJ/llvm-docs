Detailed Description

A name of a symbol.

Symbol's name can be composed of multiple strings. For example, Objective-C methods can contain multiple argument labels:

- (void) myMethodNamePiece: (int)x anotherNamePieces:(int)y;
// ^~ string 0 ~~~~~ ^~ string 1 ~~~~~
