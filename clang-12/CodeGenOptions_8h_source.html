115 // This field stores one of the allowed values for the option
116 // -fbasic-block-sections=. The allowed values with this option are:
117 // {"labels", "all", "list=<file>", "none"}.
118 //
119 // "labels": Only generate basic block symbols (labels) for all basic
120 // blocks, do not generate unique sections for basic blocks.
121 // Use the machine basic block id in the symbol name to
122 // associate profile info from virtual address to machine
123 // basic block.
124 // "all" : Generate basic block sections for all basic blocks.
125 // "list=<file>": Generate basic block sections for a subset of basic blocks.
126 // The functions and the machine basic block ids are specified
127 // in the file.
128 // "none": Disable sections/labels for basic blocks.
129 std::string BBSections;
