Detailed Description

Declaration context for names declared as extern "C" in C++.

This is neither the semantic nor lexical context for such declarations, but is used to check for conflicts with other extern "C" declarations. Example:

namespace N { extern "C" void f(); } // #1
void N::f() {} // #2
namespace M { extern "C" void f(); } // #3
N
auto * N
Definition BuildTree.cpp:1601

The semantic context of #1 is namespace N and its lexical context is the LinkageSpecDecl; the semantic context of #2 is namespace N and its lexical context is the TU. However, both declarations are also visible in the extern "C" context.

The declaration at #3 finds it is a redeclaration of N::f through lookup in the extern "C" context.
