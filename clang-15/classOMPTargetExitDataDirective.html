Detailed Description

This represents '#pragma omp target exit data' directive.

#pragma omp target exit data device(0) if(a) map(b[:])

In this example directive '#pragma omp target exit data' has clauses 'device' with the value '0', 'if' with condition 'a' and 'map' with array section 'b[:]'.
